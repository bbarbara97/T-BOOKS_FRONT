Sobre nosotros

Somos el equipo 1, formado el día 5 de agosto del 2022. Empezamos el proyecto con la planificación de la base de datos el 12 de agosto.

Seguidamente, cada semana fuimos haciendo tareas para este proyecto. En una creamos la api usando Spring y la siguiente creamos la parte de seguridad dentro de la misma.

A partir de ahí ya pudimos empezar la parte frontal del proyecto, momento en el qual escribimos esto que estas leyendo.

Por último, realizamos la integración entre la parte trasera y frontal, dando resultado al primer producto viable de todo el proyecto.
